# LIA+ AgriFood

Site institucional da frente de pesquisa LIA+ AgriFood, parte da rede LIA+ (ESALQ/USP), focada em Inteligência Artificial e Tecnologias Exponenciais aplicadas a Sistemas Agroalimentares.

🌐 **Site:** [lia-plus.github.io/agrifood](https://lia-plus.github.io/agrifood)

## Sobre este repositório

Este repositório contém o código-fonte do sub-site do LIA+ AgriFood, construído com Jekyll e tema Cayman via GitHub Pages.

## Estrutura

agrifood/
├── index.md          # Página principal do laboratório
├── _config.yml       # Configurações de rota do Jekyll
└── assets/           # Regras de estilo CSS personalizadas

## Contribuindo

Commits seguem a convenção do projeto principal:

- `feat` — nova página ou seção
- `content` — alteração de texto existente
- `assets` — imagens e arquivos estáticos
- `style` — CSS e layout
- `fix` — correções de links, imagens ou CSS
- `config` — _config.yml e configurações do Jekyll
- `docs` — este README
- `chore` — manutenção geral
